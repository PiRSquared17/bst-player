BST Player API provides the Base API for adding common web media players such as Windows Media Player, QuickTime® player, VLC Media Player, Flash® player, DivX® Web Player plugins as well as HTML 5 media to GWT applications much natural like other GWT widgets.

The API also feature a module to embed YouTube™ videos and classes to facilitate custom HTML based players much like media player skins.
