This application showcases the various media player widgets available with the BST Player API to embed multimedia files in GWT applications. It is a playground of interacting with various audio/video formats using browser plug-ins.

Using the application

The tab-pane at the bottom of the page contains the Playlist and Player Log tabs. Click any of the tabs to expand the pane and click an active tab collapse it.

Use the Playlist tab to add/remove media URLs to/from the players' playlist.

All debug messages from the application and player widgets are logged with the Player Log tab.

 

Using the Player Widgets
 
On the Player Widgets view, select the desired player provider and player widget you wish to use. Click Load Player to put the widget on the center pane of the application.
 
The Player Controls contains player properties that can be modified in-place. Enable or disable any property to see its effect.
